# Marpとは
## 存在意義の問い直し

PowerPointが重すぎてPCややる気が離陸しそうな時に使う**最終兵器**です。
テキストエディタだけでスライドが作れるため、コードを書いているふりをして*プレゼン準備*が可能です。

- **メリット**: `Git` で管理できる（コンフリクトで死ぬ）。
- **デメリット**: デザインに凝り始めると**沼**。
  - スペース2個のネストで、人生の深淵を覗くことになります。

1〜3個の `*` で囲んで、*強調*、**激怒**、***発狂*** を表現できます。

# 環境構築
## 儀式の手順

以下のコマンドをターミナルに打ち込み、ハッカー気分を味わいましょう。
実行しても、あなたの**社会的評価**は上がりません。

```bash
# 依存関係（という名のしがらみ）をインストール
$ npm install -g life-balance-destroyer

# サーバー起動
$ marp . --server

# [INFO] サーバーが立ち上がりました。
# [WARN] プレゼンの内容は空っぽです。
```

アップデートは `npm update -g` ですが、昨日のコードは動かなくなります。

# 変換オプション
## 現実逃避のフォーマット

```bash
# 上司への言い訳用（PDF）
marp index.md --pdf --allow-local-files

# パワポ絶対主義者への供物（PPTX）
marp index.md --pptx

# Webでイキる用（HTML）
marp index.md --html
```

`--pdf` をつけると、レイアウト崩れの恐怖から解放され、印刷できない恐怖が始まります。

# 数学的証明
## 世界の真理

MathJaxで数式が書けます。これで理系アピールも完璧です。

プロジェクトの成功率 $P$ は、以下の式で近似されます。

$$ P = \lim_{t \to \text{deadline}} \frac{\text{Caffeine}^2}{\text{Bugs}} $$

ここで $t$ は残り時間を表しますが、常に足りません。
